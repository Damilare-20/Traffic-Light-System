     Traffic Light System (Responsive with TailwindCSS)

            Overview
This project is a simple Traffic Light System built with HTML, JavaScript, and TailwindCSS.  
It allows users to type a traffic light color (`red`, `yellow`, or `green`) and then displays a circular light with the corresponding color along with an action message.
The design is responsive, it adapts to different screen sizes (mobile, tablet, desktop) modern look.


                 Features
Input box for user to type a color (`red`, `yellow`, or `green`)
Submit button to trigger the traffic light
Circle light that changes color based on user input


                  Action messages:
Red → "Stop! Wait until the light changes."
Yellow → "Get ready to move."
Green → "Go! You can move now."
Hidden light by default (only shows after clicking Submit)
Responsive design using TailwindCSS
